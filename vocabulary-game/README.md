# Vocabulary Recall Game

A static vocabulary practice site. Open `index.html` in a browser to run it.

## Features
- Headway Pre-Intermediate wordlist: 791 words
- Unit filter
- Three focused game modes: Smart Meaning Card, Definition → Type the word, and Sentence Gap Mode
- Unit Run Challenge page for a faster game-like practice session
- Default mastery target: 2 correct answers
- A word leaves the active game after it reaches the target correct count
- LocalStorage saves scoreboard and progress
- Add custom words from a separate Add Words tab
- Bulk import with CSV or JSON
- Filter the word scoreboard by unit, use the sort dropdown, or click table headers to sort
- Export progress and custom words

## CSV example
```csv
english,partOfSpeech,phonetics,example,unit,definition
reliable,adj,/rɪˈlaɪəb(ə)l/,She is a reliable friend.,7,A person you can trust.
```
